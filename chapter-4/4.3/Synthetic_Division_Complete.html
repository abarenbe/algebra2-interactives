What is Synthetic Division?

A Shortcut for Dividing Polynomials

Synthetic division is a quick method for dividing a polynomial by a linear factor of the form (x - k).

Instead of writing out all the variables and powers, we only work with the coefficients!

When Can You Use Synthetic Division?

Only when dividing by (x - k) where the coefficient of x is 1.

  • ✓ Dividing by (x - 3) → use k = 3
  • ✓ Dividing by (x + 5) → use k = -5 (since x + 5 = x - (-5))
  • ✗ Cannot use for (2x - 1) or (x² - 4)

The Process

1

Set Up

Write k on left, coefficients across top

2

Bring Down

Bring down the first coefficient

3

Multiply

Multiply by k, write in next column

4

Add

Add the column, write result below

5

Repeat

Continue until all columns done

Reading the Answer

The bottom row gives you:

  • All numbers except the last = coefficients of the quotient (degree is one less than dividend)
  • The last number = remainder

Example: If dividend is degree 3 and bottom row is [2, -1, 3, 5]:

  • Quotient: 2x² - x + 3
  • Remainder: 5

Finding the Right k Value

Divisor Rewrite As k Value
(x - 2) Already in form (x - k) k = 2
(x + 4) (x - (-4)) k = -4
(x - 7) Already in form (x - k) k = 7
(x + 1) (x - (-1)) k = -1

Don't Forget Zero Coefficients!

If any power of x is missing, use 0 as a placeholder.

Example: x³ + 5 has coefficients [1, 0, 0, 5] (missing x² and x terms)
