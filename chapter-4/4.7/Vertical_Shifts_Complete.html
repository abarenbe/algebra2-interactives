Understanding Vertical Shifts

What is a Vertical Shift?

A vertical shift moves the entire graph of a function up or down without changing its shape. When we add or subtract a constant to the entire function, every point on the graph moves vertically by that amount.

g(x) = f(x) + k

How It Works

  • k > 0: The graph shifts UP by k units
  • k < 0: The graph shifts DOWN by |k| units
  • k = 0: No shift (original function)

Why It Works

When we write g(x) = f(x) + k, we're adding k to every output value of f(x). If a point (a, b) is on the graph of f(x), then the point (a, b + k) is on the graph of g(x). The x-coordinate stays the same (no horizontal movement), but the y-coordinate increases by k (vertical movement).

Example: If f(2) = 5, then g(2) = f(2) + 3 = 5 + 3 = 8. The point (2, 5) moves to (2, 8).

Key Properties

  • The shape of the graph stays exactly the same
  • All x-intercepts move vertically
  • The domain remains unchanged
  • The range shifts by k units
  • Maximum and minimum values shift by k units

Real-World Application

Imagine tracking temperature throughout the day. If the temperature pattern follows some function f(t), but the next day is consistently 5 degrees warmer at every time, the new temperature function would be g(t) = f(t) + 5. The pattern is the same, just shifted up by 5 degrees.
