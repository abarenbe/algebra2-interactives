4.4 Factoring Polynomials

Master essential factoring techniques step by step

About This Section

A polynomial is factored completely when it's written as a product of unfactorable polynomials with integer coefficients. In this section, you'll learn four key factoring techniques:

  • GCF Factoring - Always the first step: factor out the greatest common factor
  • Sum & Difference of Cubes - Special patterns for a³ + b³ and a³ - b³
  • Factor by Grouping - Split 4-term polynomials into pairs
  • The Factor Theorem - Connect factors, zeros, and polynomial evaluation

Each topic includes Learn → See → Practice → Do sections to build your understanding from concepts to independent mastery.

1GCF Factoring

Factor out the greatest common factor from polynomial terms. This is always the first step in any factoring problem!

ab + ac = a(b + c)
Learn See Practice Do

2Sum & Difference of Cubes

Factor expressions of the form a³ + b³ and a³ - b³ using special patterns. Remember SOAP!

a³ ± b³ = (a ± b)(a² ∓ ab + b²)
Learn See Practice Do

3Factor by Grouping

Factor 4-term polynomials by grouping pairs of terms that share common factors.

ra + rb + sa + sb = (r + s)(a + b)
Learn See Practice Do

4The Factor Theorem

Connect factors, zeros, and polynomial evaluation. If f(k) = 0, then (x - k) is a factor!

(x - k) is a factor ⟺ f(k) = 0
Learn See Practice Do
